Data Privacy Information

This page does utilize the browser's local storage to locally save the scene and user preferences (like language settings) in the user's browser. This is done, so that the user does not lose information when closing the browser application. If you wish to delete this information simply clear your browser's cached web data.

No data is send to any remote server and no other user related data is processed by this page.
